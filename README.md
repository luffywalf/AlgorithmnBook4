###### 2.5 应用

稳定性：如果一个排序算法能够保留数组中重复元素的相对位置则可以被称为是稳定的

一部分算法是稳定的（插入，归并）但很多不是（选择，希尔，快速，堆）有很多算法能将任意算法变成稳定的，但还是蛮麻烦的
